# Loois Android SDK

此仓库是 [Loois团队](https://github.com/LOOIS-IO) 提供的方便开发者接入钱包的SDK. Loois Android SDK提供了常规的以太坊钱包功能和基于[路印协议](https://loopring.org/) Relay的交易等功能及便捷API.

提供功能如下

### 钱包

##### 创建钱包

* 生成助记词
* 派生地址路径

##### 导入钱包

* 助记词导入
* Keystore导入
* 私钥导入

##### 导出钱包

### 交易

##### 转账

* ETH转账
* ERC-20代币转账

##### 代币授权

##### 下单

##### 取消订单

##### ETH兑换WETH

### 其他

##### 绑定空投

### API
